# Dzyga's Paw — Brand Assets

This repository contains official brand assets for **Dzyga's Paw**.

For full brand guidelines see [`Dzyga'sPaw_logobook.pdf`](Dzyga'sPaw_logobook.pdf).

---

## Logo Files

Assets are organized by logo type, then format.

```
logos/
├── primary/        # Full horizontal logo (wordmark + paw)
│   ├── svg/
│   └── png/
├── secondary/      # Stacked / alternative layout
│   ├── svg/
│   └── png/
└── icon/           # Paw icon only
    ├── svg/
    └── png/
```

### Variants

Each logo type comes in four variants:

| Variant | Filename suffix | Use on |
|---|---|---|
| Dark | `_Dark` | Light backgrounds |
| Light | `_Light` | Dark backgrounds |
| Mono Black | `_MonoBlack` | Single-color print, dark |
| Mono White | `_MonoWhite` | Single-color print, light |

> **Note:** Primary logo only has Dark and Light variants (no mono).

## Quick Picks

- **Website / app (light bg):** `logos/primary/svg/PrLogo_Dark.svg`
- **Dark mode / dark bg:** `logos/primary/svg/PrLogo_Light.svg`
- **Favicon / avatar:** `logos/icon/png/IconLogo_Dark.png`
- **Monochrome print:** `logos/secondary/svg/SecLogo_MonoBlack.svg`

---

## File Format Guide

- **SVG** — preferred for web, UI, and print. Infinitely scalable.
- **PNG** — use when SVG isn't supported (social media, presentations, etc.).

---

## Usage Notes

- Do not distort, recolor, or alter the logo.
- Maintain clear space around the logo equal to the height of the paw icon.
- When in doubt, refer to the full logobook PDF.
